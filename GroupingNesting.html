Grouping

When several selectors share the same declarations, they may be grouped into a comma-separated list.

Example

Below, you can see how 3 different selectors with the same declarations can be grouped into a comma-separated list:

  • Non-grouped list:
    •

    h1 { font-family: sans-serif }
    h2 { font-family: sans-serif }
    h3 { font-family: sans-serif }

  • Grouped list:
    •

    h1, h2, h3 { font-family: sans-serif }

    Both examples above will yield the same results.

    Nesting

    Nesting allows you to specify properties to selectors within other selectors.

    Example

    #top { background-color: #ccc; padding: 1em }
    #top h1 { color: #ff0; }
    #top p { color: red; font-weight: bold; }

    The above code removes the need for classes or IDs on the <p> and <h1> tags if it is applied to HTML that looks something like this:

    <div id="top">
    <h1>Chocolate curry</h1>
    <p>This is my recipe for making curry purely with chocolate</p>
    <p>Mmm mm mmmmm</p>
    </div>
