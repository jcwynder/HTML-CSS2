Flexbox

The flexbox layout is one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit into smaller spaces.

A flexible layout must have a parent element with the display property set to flex.

Direct child elements of the flexible container automatically becomes flexible items.

To start using the Flexbox model, you need to first define a flex container. The flex container becomes flexible by setting the display to flex.
